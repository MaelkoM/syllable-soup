About Syllable Soup

The why and the how.

This app generates unique names that are inspired by a large set of actual names.
What separates this generator from others is that the generated names aren't just copies of existing ones but (more or less) completely new.

Soon-to-be parents, programmers, con artists - anyone searching for names that don't exist (yet).

I tried tree methods with varying results.
All methods use a large list with real names to generate new ones.
The list I used was compiled by c't. It can be downloaded here.

How it works

Using string operations (later versions will be updated to regex), the previously saved names are split into syllables (first syllables, last syllables and middle syllables).
After dissecting the names, random first and last syllables are selected and random middle syllables are stuffed between them.

Pros and cons

    • The generated names are okay but more on the adventurous side.
    • The code is quick, so you can generate vast amounts of names in a few seconds.

How it works

The real names ar cut into even parts (e.g. pairs of letters). Those are called "states".
Knowing which states normally follow one another in those real names, the probabilities for each state change (aka transition probabilities) are calculated.

After that, the states are combined respecting the determined probabilities.

Pros and cons

    • The generated names are quite good.
    • Depending on how the transition probabilities are calculated and stored, there are two choices:
      • The code takes a very long time to initialize
      • The generation of each name takes a little bit longer.

How it works

In a generative adversarial network (GAN), we are pitting two neural networks against each other.
One is called the generator and the other one ist called the discriminator.

Here, the Generator tries to recreate names from random samples of letters while the discriminator rejects the generated names it thinks are not real.
Both networks learn from ech other, so the quality of the output will improve over time.

If the networks were set up properly, that is...
